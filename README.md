# vue-pseudo
> Emulate native pseudo class effect.

## Usage
```js
import VuePseudo from 'vue-pseudo';
Vue.use(VuePseudo);
```

* `1.x.x` for `Vue@1.x.x`.
* `2.x.x` for `Vue@2.x.x`.

## :active
```html
<div v-pseudo:active="'is-active'"></div>
```
